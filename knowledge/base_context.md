## Output Conventions
- JSON 输出：必须可被 `JSON.parse()` 直接解析，无额外文本
- Markdown 输出：What/Why/How/Result/Next 五段式
- 语言：与输入保持一致，不主动翻译
- 数值/指标必须可验证，不可虚构
