## 环境准备

1. Python 3.9+ 运行环境
2. 满足以下技术栈要求：
- Python 3.9+
- Markdown
- JSON
- Git

## 安装步骤

```bash
# 1. 克隆项目
git clone <repository_url>
cd BOS-FS

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量（如需要）
cp .env.example .env
```
